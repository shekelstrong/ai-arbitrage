# AI Arbitrage

Современный Next.js сайт для русскоязычной аудитории про заработок на AI и партнёрских программах.

## Структура
- **Главная** — лендинг с преимуществами, отзывами и призывом к действию
- **/articles/** — блог со статьями про AI, заработок и инструменты
- **/products/** — обзоры продуктов и сервисов для арбитража
- **/about/** — информация о проекте

## Технологии
- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- TypeScript
- Статическая генерация (SSG)

## SEO
- sitemap.xml и robots.txt
- Open Graph и Twitter Card
- Schema.org Article разметка
- Мета-теги на русском для Яндекса

## Запуск
```bash
npm install
npm run dev
```
